PrivacyGuard 脱敏卫士

智能文档脱敏工具，一键识别并遮盖敏感信息，所有处理本地完成，确保您的数据安全

完全免费 本地处理 开源安全 跨平台支持

核心功能

身份证识别

自动识别 18 位中国身份证号，支持 15 位旧版号码，智能匹配避免误伤

手机号检测

精准识别 11 位中国大陆手机号码，支持多种号码段自动匹配

邮箱识别

智能匹配各类邮箱格式，有效防止邮件地址信息泄露风险

银行卡号

识别 16-19 位银行卡号，全面保护金融敏感信息安全

本地处理

所有文档在本地完成处理，不上传云端，最大程度保护数据隐私

OCR 智能识别

图片版 PDF 使用 OCR 技术自动识别文字，精确检测敏感信息

安装指南

方式一：DMG 安装（推荐）

1
下载安装包

从官网或发布渠道下载 PrivacyGuard-v37.0.10-macOS.dmg

2
打开 DMG 文件

双击下载的 DMG 文件，会弹出安装窗口，显示 PrivacyGuard 图标和 Applications 文件夹

3
安装应用

将 PrivacyGuard 图标拖拽到 Applications 文件夹，等待复制完成

4
首次运行

打开「启动台」点击 PrivacyGuard。如提示「无法验证开发者」，请前往「系统设置」→「隐私与安全性」→「仍要打开」

安全提示

macOS 可能会阻止未认证开发者应用运行，这是正常现象，请在系统设置中授权。

方式一：安装程序（推荐）

1
下载安装包

下载 PrivacyGuard-v37.0.10-Setup.exe

2
运行安装程序

双击安装程序，按照向导提示完成安装。可选择安装路径（默认：C:\Program Files\PrivacyGuard）

3
完成安装

可选择立即运行，桌面会创建快捷方式

方式二：便携版

直接运行 PrivacyGuard.exe，无需安装，可放在 U 盘中随身携带。

打开 PDF

保存文件

2
点击「智能脱敏」

系统自动识别敏感信息，用红色方框标记检测到的区域。对于图片版 PDF，使用 OCR 技术识别

3
调整脱敏区域

添加、删除或调整脱敏框位置和大小，确保覆盖所有敏感信息

4
保存脱敏文件

点击「保存」按钮，选择保存位置，文件名自动添加 _脱敏 后缀

Word 脱敏流程

打开 Word
选择模式
执行脱敏
保存文档

智能模式

点击「智能脱敏」自动扫描所有敏感信息，一键全部脱敏

精确模式

选中文字 → 右键「脱敏选中内容」→ 只脱敏选中的这一处

对于图片版 PDF（扫描件），系统使用 RapidOCR 技术识别文字。

普通 (1.0x)

清晰文档，追求速度

默认推荐

标准 (1.5x)

高精 (2.0x)

脱敏框操作

操作 方法 说明
添加框 点击「添加」+ 拖拽绘制 在页面上绘制矩形区域
删除框 Delete 或点击「删除」 删除选中的脱敏框
移动框 拖拽框体 改变脱敏框的位置
调整大小 拖拽控制点 8 个方向可调整
精确调整 工具栏偏移设置 X 偏移、宽度微调

批量图片处理

支持批量选择图片

点击「打开 PDF」按钮，在文件对话框中多选图片（按住 Ctrl/Cmd），系统自动将所有图片合并为单个 PDF 进行处理。

Word 处理功能

三种脱敏模式

智能模式

自动扫描整篇文档，识别所有敏感信息，可一键全部脱敏或逐条确认

精确模式

只脱敏选中的特定文字，不影响其他位置的相同文字，精确控制每一个脱敏点

全局模式

脱敏文档中所有相同的文字，一次性批量处理，适合需要全局替换的场景

撤销功能

撤销单个脱敏

在精确模式下，点击已脱敏的区域，右键选择「撤销脱敏」，只有该处的脱敏被撤销

撤销全部脱敏

点击「撤销全部脱敏」按钮，所有脱敏操作被撤销，文档恢复到初始状态

位置保持功能

脱敏操作后，滚动位置保持不变，方便继续编辑同一区域。

配置系统

配置文件位置

PrivacyGuard/
├── PrivacyGuard.exe (Windows) 或 PrivacyGuard.app (macOS)
├── config.json              ← 用户配置文件（自动生成）
└── config.json.template     ← 配置模板参考

常用配置项

配置项 说明 默认值
app.name 应用名称 PrivacyGuard 脱敏卫士
redaction.replacement_text 脱敏替换文本 [已脱敏]
redaction.scan.default_level 默认扫描精度 1.0
ocr.accuracy.default 默认 OCR 精度 1.0

配置文件示例

{
  "app": {
    "name": "PrivacyGuard 脱敏卫士",
    "window": {
      "default_width": 1300,
      "default_height": 900
    }
  },
  "redaction": {
    "replacement_text": "[已脱敏]",
    "scan": {
      "default_level": 1.0,
      "available_levels": [1.0, 1.5, 2.0]
    }
  },
  "ocr": {
    "accuracy": {
      "default": 1.0
    }
  }
}

常见问题

PrivacyGuard 是免费的吗？

是的，PrivacyGuard 完全免费使用，没有功能限制，也没有广告。

脱敏后的文件会保存到云端吗？

不会。所有处理都在本地完成，文件不会上传到任何服务器，确保您的数据安全。

可以自定义替换文本吗？

可以。通过修改 config.json 中的 redaction.replacement_text 配置项，例如改为 ****** 或其他文本。

如何处理识别不准确的 PDF？

1. 在 OCR 设置中选择更高的精度级别（高精 2.0x）
2. 手动添加或调整脱敏框位置
3. 使用偏移调整功能微调位置
4. 对于手写体，建议手动添加脱敏框

支持哪些语言的 OCR 识别？

主要支持中文和英文。其他语言的识别效果可能不理想。

为什么 Windows 上会报毒？

PyInstaller 打包的程序可能被杀毒软件误报，这是因为程序没有数字签名。您可以添加到杀毒软件白名单，或从官方渠道下载确保安全。

故障排除

应用无法启动

解决方案

1. 检查系统版本是否符合要求
2. macOS：前往「系统设置」→「隐私与安全性」授权
3. Windows：尝试以管理员身份运行
4. Windows：检查是否有 VC++ Redistributable

PDF 打开失败

1
验证文件

确认文件没有损坏，尝试用其他 PDF 阅读器打开验证

2
检查占用

检查文件是否被其他程序占用，关闭其他 PDF 程序后重试

3
重启应用

关闭 PrivacyGuard 后重新打开

OCR 识别不准确

提高精度

设置 → 扫描精度 → 选择「高精 (2.0x)」

手动添加

手动添加遗漏的脱敏框，确保覆盖所有敏感信息

.doc 文件无法打开

解决方案

1. 确认已安装 LibreOffice
2. 安装后重启 PrivacyGuard
3. 如果仍有问题，先用 Word 另存为 .docx 格式再打开
