## 项目概述

**名称**: PrivacyGuard 脱敏卫士
**类型**: PDF 隐私信息脱敏工具
**语言**: Python 3 + PyQt6
**用途**: 智能识别并脱敏 PDF 文档中的敏感信息

6. 点击 "💾 导出" 保存结果
