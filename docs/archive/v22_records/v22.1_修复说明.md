# PrivacyGuard v22.1 修复说明

## 📋 本次修复内容

### 问题
- 右键点击脱敏框完全无效
- 控制台没有任何 [DEBUG] 日志输出

### 根本原因
QScrollArea 的控件层级结构：
```
MainWindow
  └── scroll (QScrollArea)
        └── viewport()  ← 鼠标事件实际到达这里
              └── canvas_container (QWidget)
```

**原问题**: v22.0 将事件过滤器安装在 `canvas_container` 上，但鼠标事件在到达 `canvas_container` 之前就被 `viewport()` 拦截了。

### 解决方案
将事件过滤器安装到 `scroll.viewport()` 上，这是真正接收鼠标事件的控件。

## 🔧 代码变更

### 1. 事件过滤器安装位置 (main.py:652)
```python
# v22.0 (错误)
self.canvas_container.installEventFilter(self)

# v22.1 (正确)
self.scroll.viewport().installEventFilter(self)
```

### 2. 事件过滤器检查 (main.py:520)
```python
# v22.0 (错误)
if obj == self.canvas_container and event.type() == event.Type.MouseButtonPress:

# v22.1 (正确)
if obj == self.scroll.viewport() and event.type() == event.Type.MouseButtonPress:
```

### 3. 版本号更新 (main.py:24)
```python
VERSION = "22.1 - Viewport Filter"
```
