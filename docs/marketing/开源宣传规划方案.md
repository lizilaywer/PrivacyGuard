## 一、项目定位与价值主张

### 1.1 项目背景

**PrivacyGuard 脱敏卫士** 是由执业律师汪立（安徽始信律师事务所）独立开发的文档隐私保护工具。作为一名**全栈律师、前教师、退伍军人**，汪立将法律实务中对信息保护的迫切需求转化为技术实践，打造出一款真正懂法律人痛点的脱敏工具。

### 1.2 核心价值

| 维度 | 价值主张 |
|------|----------|
| **专业深度** | 由执业律师设计，深度契合法律实务场景 |
| **技术广度** | Python + PyQt6 + OCR，支持PDF/Word双格式 |
| **使用零门槛** | 可视化操作，3分钟上手 |
| **隐私零泄露** | 本地运行，数据不上传云端 |
| **成本零负担** | 完全免费开源，无功能限制 |

### 1.3 差异化优势

```
传统脱敏工具          PrivacyGuard
    │                    │
    ▼                    ▼
┌──────────┐        ┌──────────────┐
│ 云端处理  │   vs   │ 本地离线运行  │ ← 律师函件绝不外泄
├──────────┤        ├──────────────┤
│ 订阅收费  │   vs   │ 永久免费开源  │ ← 降低法律人工作成本
├──────────┤        ├──────────────┤
│ 技术门槛高 │   vs   │ 可视化操作   │ ← 书记员也能轻松使用
├──────────┤        ├──────────────┤
│ 仅支持PDF │   vs   │ PDF+Word双格式│ ← 适配律所实际工作流
└──────────┘        └──────────────┘
```

## 二、目标用户群体分析

### 2.1 核心用户画像

#### 画像一：执业律师（主力用户）
- **年龄段**: 28-45岁
- **技术能力**: 中等，习惯使用办公软件
- **痛点**: 案件材料脱敏耗时、担心信息泄露、需要快速交付
- **使用场景**: 案例分享前的敏感信息处理、对外提交材料脱敏

#### 画像二：法务人员
- **年龄段**: 25-40岁
- **技术能力**: 中等到较高
- **痛点**: 批量合同脱敏、合规审查材料准备
- **使用场景**: 合同归档、外部审计材料准备

#### 画像三：司法行政人员
- **年龄段**: 30-50岁
- **技术能力**: 中等偏下
- **痛点**: 公开文书脱敏、信息公开材料处理
- **使用场景**: 裁判文书公开、政务信息公开

#### 画像四：法律科技公司
- **技术能力**: 高
- **痛点**: 需要可定制的脱敏组件
- **使用场景**: 集成到自有法律科技产品中

### 2.2 用户需求优先级

| 优先级 | 需求 | 对应功能 |
|--------|------|----------|
| P0 | 一键智能识别 | OCR + 正则双引擎 |
| P0 | 精确手动脱敏 | 选中即脱敏，避免过度处理 |
| P1 | Word格式支持 | 律所主要工作文档格式 |
| P1 | 批量撤销 | 误操作快速恢复 |
| P2 | 自定义规则 | 适配特殊敏感信息类型 |

#### 中版本（200字）

> PrivacyGuard 脱敏卫士由执业律师独立开发，深度理解法律实务中的隐私保护需求。支持PDF和Word文档的智能脱敏，采用OCR+正则双引擎识别身份证号、手机号、邮箱等敏感信息，同时提供精确模式和全局模式的手动脱敏功能。所有处理均在本地完成，确保案件材料绝不外泄。永久免费开源，让每位法律人都能零成本拥有专业的文档脱敏能力。

#### 长版本（500字）

> **从法律实务中诞生的脱敏工具**
>
> 作为一名执业律师，我深知在案例研究、学术交流、业务分享中，文档脱敏是一个高频但痛苦的工作。传统的处理方式要么依赖云端服务（存在信息泄露风险），要么需要复杂的命令行工具（技术门槛过高）。
>
> **PrivacyGuard 脱敏卫士** 应运而生——这是一款专为法律人设计的免费开源文档脱敏工具。
>
> **核心能力**
> - 🔍 **智能识别**：OCR + 正则双引擎，自动识别身份证号、手机号、邮箱、银行卡号等敏感信息
> - 📝 **双格式支持**：同时支持PDF和Word文档，覆盖律所全部工作场景
> - 🎯 **精确脱敏**：只脱敏选中内容，避免一刀切的过度处理
> - 🔄 **批量撤销**：误操作一键恢复，支持全局批量撤销
> - 💻 **本地运行**：数据不出本机，彻底杜绝云端泄露风险
>
> **技术亮点**
> - Python + PyQt6 跨平台架构，支持macOS和Windows
> - RapidOCR 本地OCR引擎，识别准确率高
> - 可视化操作界面，3分钟上手零门槛
>
> **开源承诺**
> 本项目采用开源协议发布，永久免费使用。欢迎法律科技爱好者参与贡献，共同打造法律人最好用的小工具。
>
> **开发者**：汪立，安徽始信律师事务所执业律师，全栈开发者，前教师，退伍军人
