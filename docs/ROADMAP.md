# PrivacyGuard 开发路线图

## 版本策略

### 版本编号规则

```
主版本号.次版本号 - 功能描述
```

- **主版本号**: 重大功能变更或不兼容更新
- **次版本号**: 新功能或重要改进
- **功能描述**: 简短说明本次更新的主要内容

## 贡献指南

欢迎贡献代码、报告 Bug 或提出功能建议！

### 如何贡献

1. Fork 项目
2. 创建功能分支 (`git checkout -b feature/AmazingFeature`)
3. 提交更改 (`git commit -m 'Add some AmazingFeature'`)
4. 推送到分支 (`git push origin feature/AmazingFeature`)
5. 开启 Pull Request

### 功能建议

在提交功能建议前，请先：
1. 搜索现有的 Issues
2. 确认功能未在路线图中
3. 提供详细的功能描述和使用场景
