┗━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━┛

**变更内容**:
  🆕 新增印章自动检测功能（OpenCV 图像处理）
  🆕 基于颜色（红色）+ 形状（圆形度）检测
  🆕 高级设置新增"印章"复选框（默认不勾选）
  ✅ 无需额外依赖（使用现有 OpenCV）

**技术实现**:
  - 使用 OpenCV HSV 色彩空间检测红色区域
  - 形态学操作去噪和连接相近区域
  - 轮廓分析：面积、宽高比、圆形度过滤
  - 红色像素占比阈值：30%
  - 圆形度阈值：0.5
  - 仅在勾选"印章"时启用

**影响组件**:
  - OCRWorker: 增加 _get_seal_detector(), _detect_seals() 方法
  - SettingsDialog: 自动生成"印章"复选框
  - 配置系统: 新增 seal_detection 配置项

**修改文件**:
  - main.py: 约 +100 行，印章检测逻辑（OpenCV 实现）
  - config.json.template: 印章配置
  - requirements.txt: 无变化（移除了 PaddleOCR 依赖）
