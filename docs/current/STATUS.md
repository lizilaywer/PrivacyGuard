**重大变更**:
  ✅ 完全移除 PaddleOCR 引擎及相关代码
  ✅ 简化 OCR 架构，只保留 RapidOCR
  ✅ 移除 OCR 引擎选择 UI
  ✅ 删除 PaddleOCR 模型文件
  ✅ 移除 paddleocr/paddlepaddle 依赖

**原因**:
  - PaddleOCR 3.4 的字符级坐标系统与项目架构存在根本性兼容问题
  - Y 轴偏移问题经过多次尝试仍无法稳定解决
  - 维护成本高，稳定性无法保证
  - RapidOCR 单引擎足以满足需求

**文件变更**:
  - main.py: 移除 OCR 引擎选择逻辑
  - privacyguard/ocr/paddleocr.py: 已删除
  - privacyguard/ocr/manager.py: 简化为单引擎
  - privacyguard/ocr/__init__.py: 移除 PaddleOCR 导出
  - config.json/template: 移除 ocr.engine 配置
  - requirements.txt: 移除 paddleocr/paddlepaddle
