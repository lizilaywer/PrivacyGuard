PrivacyGuard 品牌图标方案

方向关键词：本地离线、法律专业、克制可信、文档脱敏。设计目标不是“炫”，而是让用户第一眼觉得可靠、现代、像桌面正式产品。

本地离线 法律场景 文档工作台 简洁克制

Logo 与 App Icon

主图标把“盾牌、文档、脱敏条”合成一个符号。意思很直接：保护文档中的敏感信息。相比单纯用 PG 字母，这个方向更像成熟工具软件。

设计判断

这版品牌视觉更适合你的软件定性：不是娱乐工具，不是 AI 聊天产品，而是给法律文档和隐私场景使用的专业桌面应用。

可信 主色用稳重的蓝色，强调安全、秩序和桌面端专业感。
易懂 核心符号是文档加脱敏条，不需要解释，用户一眼能懂。
可延展 同一套线条和圆角规则，可以自然扩展到功能图标、安装包图标和启动页。

功能图标

这一套是给首页四大功能、工具栏和未来帮助页准备的图标语言。统一的做法是：浅底、圆角、低饱和背景，主信息放在图标中心。

PDF 脱敏

文档页 + 脱敏条。代表 PDF 内的敏感信息遮盖，语义直接。

Word 替换

文档页 + 替换箭头。表达“内容替换”，区别于遮罩式脱敏。

批量 Word

叠放文档，强调一套规则同时应用到多个文件。

图片合并 PDF

小图片汇入单页文档，表达“先整理图片，再生成 PDF”。

高级设置

三条滑杆，延续“精调参数”的视觉暗示，比齿轮更现代。

使用 / 反馈

气泡 + 提问感，更适合帮助、反馈、手册这类支持区入口。
