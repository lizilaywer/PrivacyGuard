性能方向 本地原生

不要把高性能预览网页化

PDF 继续用原生渲染画布，Word 保留 QWebEngine 双栏预览，但外壳和面板全部重构。 这样既能获得现代外观，也不会牺牲 Windows 上的启动速度和内存表现。
