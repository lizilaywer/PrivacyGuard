# PrivacyGuard 脱敏卫士 Logo 设计规范

> 参考阿里巴巴、腾讯等国内大厂软件图标设计标准
> 版本: v1.0 | 日期: 2026-03-17

---

## 一、设计理念

### 1.1 设计关键词
- **安全**: 盾牌、保护、信任
- **隐私**: 隐藏、加密、敏感信息处理
- **专业**: 简洁、现代、办公软件气质
- **亲和力**: 圆润、友好、不冰冷

### 1.2 设计方向
参考钉钉（阿里）、企业微信（腾讯）、飞书的现代扁平化风格：
- 圆角矩形轮廓（ macOS 圆角更大，Windows 略方正）
- 渐变色营造层次感
- 简洁图形符号，避免过多细节
- 小尺寸下依然清晰可辨

## 二、色彩规范

```
主色: #2563EB (Blue-600)
渐变起点: #3B82F6 (Blue-500)
渐变终点: #1D4ED8 (Blue-700)

### 2.2 辅助色
```
安全绿: #10B981 (Emerald-500) - 用于"保护中"状态

|------|------|------|------|

## 四、图形设计规范

### 4.1 主图标构成
```
┌─────────────────────────┐
│    圆角矩形背景          │
│                         │
│         ┌───┐           │
│         │ P │           │
│         │ G │           │
│         └───┘           │
│                         │
│                         │
└─────────────────────────┘
```

### 4.2 元素规范
1. **圆角矩形背景**: 蓝色渐变，圆角 200px（1024px 画布）
2. **PG 字母**: 大号白色文字，系统字体，加粗
3. **简洁设计**: 无多余装饰，各尺寸清晰可辨

### 4.3 安全区域
- 图标主要内容距离边缘 ≥ 10%
- 避免内容触及圆角边界

```

- logo_dark_512.png
- logo_mono_256.png
```

## 六、实现方案

### 6.1 技术栈
- **设计工具**: Figma / Adobe Illustrator（设计阶段）
- **矢量格式**: SVG（主源文件）
- **栅格格式**: PNG（透明背景）
- **打包格式**: ICO (Windows) / ICNS (macOS)
- **生成脚本**: Python + Pillow + CairoSVG

### 6.2 生成流程
1. 设计并导出 SVG 主源文件
2. 使用 Python 脚本批量生成各尺寸 PNG
3. 使用 icotool (Linux) 或 Python 库生成 ICO
4. 使用 iconutil (macOS) 或 png2icns 生成 ICNS

## 七、使用说明

### 7.1 打包时引用

**Windows (PyInstaller)**:
```python
# .spec 文件
exe = EXE(
    ...
    icon='assets/logo/windows/app_icon.ico',
    ...
)
```

**macOS (PyInstaller)**:
```python
# .spec 文件
app = BUNDLE(
    ...
    icon='assets/logo/macos/AppIcon.icns',
    ...
)
```

### 7.2 应用内引用
```python
# 加载图标
from PyQt6.QtGui import QIcon
from PyQt6.QtCore import QDir

icon_path = QDir.current().filePath('assets/logo/export/256/logo_default_256.png')
app.setWindowIcon(QIcon(icon_path))
```

## 八、参考案例

### 8.1 阿里系风格参考
- **钉钉**: 蓝色渐变，圆角方形，简洁对话气泡
- **语雀**: 绿色渐变，书本+羽毛笔意象
- **蚂蚁森林**: 绿色系，扁平化树形图标

### 8.2 腾讯系风格参考
- **企业微信**: 蓝绿色渐变，对话气泡+勾选
- **腾讯文档**: 蓝色，文档+协作元素
- **腾讯会议**: 蓝紫色渐变，摄像头意象

### 8.3 PrivacyGuard 差异化
- 比钉钉更稳重（安全工具属性）
- 比企业微信更专业（聚焦文档处理）
- 融入"隐私保护"的视觉暗示（盾牌+马赛克）
