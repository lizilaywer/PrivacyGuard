## 一、产品简介

### 1.1 什么是 PrivacyGuard？

PrivacyGuard 脱敏卫士是一款专业的文档脱敏工具，专为需要处理敏感信息的用户设计。它可以自动识别并遮盖文档中的敏感信息，包括：

- 🔢 **身份证号** - 自动识别 18 位身份证号
- 📱 **手机号** - 识别中国大陆手机号码
- 📧 **邮箱地址** - 匹配各类邮箱格式
- 💳 **银行卡号** - 识别 16-19 位银行卡号
- 📝 **自定义规则** - 支持自定义正则表达式

### 1.2 支持格式

| 格式 | 支持情况 | 说明 |
|------|----------|------|
| PDF | ✅ 完全支持 | 文字版和图片版 PDF |
| Word (.docx) | ✅ 完全支持 | Office 2007+ 格式 |
| Word (.doc) | ✅ 支持 | 旧版 Word 格式 |

### 1.3 系统要求

**macOS**
- macOS 11.0 (Big Sur) 或更高版本
- Intel 或 Apple Silicon (M1/M2/M3)
- 4GB 内存（推荐 8GB）
- 500MB 可用磁盘空间

**Windows**
- Windows 10 或 Windows 11
- 64 位系统
- 4GB 内存（推荐 8GB）
- 500MB 可用磁盘空间

## 二、安装指南

### 2.1 macOS 安装

#### 方式一：DMG 安装（推荐）

1. **下载安装包**
   - 从官网或微信群下载 `PrivacyGuard-37.0-macOS.dmg`

2. **打开 DMG 文件**
   - 双击下载的 DMG 文件
   - 会弹出安装窗口，显示 PrivacyGuard 图标和 Applications 文件夹

3. **安装应用**
   - 将 PrivacyGuard 图标拖拽到 Applications 文件夹
   - 等待复制完成

4. **首次运行**
   - 打开「启动台」，点击 PrivacyGuard
   - 如果提示「无法打开，因为无法验证开发者」：
     - 前往「系统设置」→「隐私与安全性」
     - 点击「仍要打开」
   - 如果提示需要访问文件夹权限，请点击「允许」

#### 方式二：便携版

直接运行解压后的 `.app` 文件，无需安装。

### 2.2 Windows 安装

#### 方式一：安装程序（推荐）

1. **下载安装包**
   - 下载 `PrivacyGuard-37.0-Setup.exe`

2. **运行安装程序**
   - 双击安装程序
   - 按照向导提示完成安装
   - 可选择安装路径（默认：`C:\Program Files\PrivacyGuard`）

3. **完成安装**
   - 可选择「立即运行 PrivacyGuard」
   - 桌面会创建快捷方式

#### 方式二：便携版

直接运行 `PrivacyGuard.exe`，无需安装，可放在 U 盘中随身携带。

## 三、快速入门

### 3.1 启动应用

双击应用图标启动 PrivacyGuard，主界面如下：

```
┌─────────────────────────────────────────────────────────────┐
│  PrivacyGuard 脱敏卫士                    [设置] [反馈] [?]  │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│                    [ 拖拽文件到这里 ]                        │
│                                                             │
│              支持 PDF、Word (.docx/.doc) 格式               │
│                                                             │
│                  或点击选择文件                              │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

### 3.2 打开文件

**方式一：拖拽**
- 直接从文件夹拖拽文件到窗口中央区域

**方式二：点击选择**
- 点击「选择文件」按钮
- 在弹出的文件浏览器中选择文件

### 3.3 PDF 脱敏流程

1. **打开 PDF 文件**
   - 拖拽或选择 PDF 文件
   - 系统自动检测 PDF 类型（文字版/图片版）

2. **自动识别**
   - 系统会自动识别文档中的敏感信息
   - 用红色方框标记检测到的区域

3. **调整脱敏区域**
   - **添加区域**：点击工具栏「添加」，在页面上拖拽绘制
   - **删除区域**：选中区域后按 Delete 或点击「删除」
   - **调整大小**：拖拽区域边缘的控制点
   - **移动位置**：拖拽区域整体

4. **执行脱敏**
   - 点击「开始脱敏」按钮
   - 选择保存位置
   - 等待处理完成

### 3.4 Word 脱敏流程

1. **打开 Word 文件**
   - 支持 .docx 和 .doc 格式
   - 文件会在内置浏览器中打开

2. **选择脱敏模式**

   **精确模式**：
   - 选中需要脱敏的文字（高亮显示）
   - 右键选择「脱敏选中内容」
   - 适合精确控制脱敏范围

   **全局模式**：
   - 点击「扫描敏感信息」
   - 系统高亮显示所有匹配的敏感信息
   - 点击每个高亮区域进行脱敏
   - 或点击「全部脱敏」一键处理

3. **保存文件**
   - 点击「保存文档」
   - 选择保存位置和文件名

## 四、功能详解

### 4.1 PDF 处理功能

#### 4.1.1 OCR 文字识别

对于**图片版 PDF**（扫描件），系统使用 OCR 技术识别文字：

- **识别精度选项**：
  - 高速度（适合清晰文档）
  - 平衡模式（推荐）
  - 高精度（适合模糊文档）

- **识别内容**：
  - 自动识别文字位置
  - 支持中英文混合
  - 自动检测身份证号、手机号等

#### 4.1.2 脱敏框操作

| 操作 | 方法 | 说明 |
|------|------|------|
| 添加框 | 工具栏「添加」+ 拖拽 | 在页面上绘制矩形 |
| 删除框 | Delete 键或工具栏 | 删除选中的框 |
| 移动 | 拖拽框体 | 改变框的位置 |
| 调整大小 | 拖拽控制点 | 8 个方向可调整 |
| 精确调整 | 工具栏偏移设置 | X 偏移、宽度微调 |

#### 4.1.3 导出选项

- **保存格式**：仅 PDF（保持原格式）
- **文件名**：自动添加 `_脱敏` 后缀
- **保存位置**：可自定义

### 4.2 Word 处理功能

#### 4.2.1 精确模式

适合需要精确控制脱敏范围的场景：

1. 用鼠标选中需要脱敏的文字
2. 选中的文字会高亮显示
3. 右键选择「脱敏选中内容」
4. 文字会被替换为 `[已脱敏]`

**特点**：
- 精确控制每一个脱敏点
- 支持批量撤销
- 保持文档格式完整

#### 4.2.2 全局模式

适合快速处理大量敏感信息：

1. 点击「扫描敏感信息」按钮
2. 系统自动识别所有敏感信息
3. 匹配的文本会高亮显示
4. 点击单个高亮区域进行脱敏
5. 或点击「全部脱敏」一键处理

**特点**：
- 快速识别整篇文档
- 自动匹配规则
- 可逐条确认或批量处理

#### 4.2.3 撤销功能

- **撤销单个**：在精确模式下，可单独撤销某次脱敏
- **撤销全部**：点击「撤销全部脱敏」恢复原始文档
- **位置保持**：撤销后滚动位置保持不变

### 4.3 设置选项

点击主界面「设置」按钮打开设置面板：

#### 4.3.1 脱敏规则

| 规则 | 默认状态 | 说明 |
|------|----------|------|
| 身份证号 | 启用 | 18 位中国身份证号 |
| 手机号 | 启用 | 11 位手机号 |
| 邮箱地址 | 启用 | 各类邮箱格式 |
| 银行卡号 | 启用 | 16-19 位银行卡号 |
| 自定义规则 | 可添加 | 支持正则表达式 |

#### 4.3.2 扫描精度

- **精度级别**：1.5x / 2.0x / 3.0x
- **建议**：
  - 清晰文档：1.5x（速度快）
  - 一般文档：2.0x（推荐）
  - 模糊文档：3.0x（精度高）

#### 4.3.3 偏移调整

用于微调 PDF 脱敏框位置：

- **X 偏移**：左右平移（-20 到 +20 像素）
- **宽度调整**：调整框的宽度（-20 到 +20 像素）
- **默认值**：0（不调整）

### 4.4 配置文件（高级）

从 v37.0 开始，PrivacyGuard 支持 JSON 配置文件：

**文件位置**：`config.json`（与主程序同目录）

**可配置项**：
- 应用名称和窗口尺寸
- 脱敏规则和替换文本
- OCR 参数
- 安全选项

**配置示例**：
```json
{
  "app": {
    "name": "PrivacyGuard 脱敏卫士",
    "window": {
      "default_width": 1300,
      "default_height": 900
    }
  },
  "redaction": {
    "replacement_text": "[已脱敏]",
    "scan": {
      "default_level": 2.0
    }
  }
}
```

## 五、配置说明

### 5.1 配置文件结构

```
PrivacyGuard/
├── PrivacyGuard.exe (Windows) 或 PrivacyGuard.app (macOS)
├── config.json              ← 用户配置文件（自动生成）
└── config.json.template     ← 配置模板参考
```

### 5.2 常用配置项

#### 应用设置
```json
{
  "app": {
    "name": "PrivacyGuard 脱敏卫士",
    "window": {
      "default_width": 1300,
      "default_height": 900,
      "min_width": 900,
      "min_height": 600
    }
  }
}
```

#### 脱敏规则
```json
{
  "redaction": {
    "default_rules": {
      "身份证号": {
        "pattern": "(?<!\\d)([1-9]\\d{5}(19|20)\\d{2}(0[1-9]|1[0-2])(0[1-9]|[12]\\d|3[01])\\d{3}[\\dXx])(?!\\d)",
        "enabled": true
      },
      "手机号": {
        "pattern": "(?<![\\d])(1[3-9]\\d{9})(?![\\d])",
        "enabled": true
      }
    },
    "replacement_text": "[已脱敏]"
  }
}
```

#### OCR 设置
```json
{
  "ocr": {
    "accuracy": {
      "low": 1.5,
      "medium": 2.0,
      "high": 3.0,
      "default": 2.0
    }
  }
}
```

### 5.3 配置热重载

修改 `config.json` 后：
- 重启应用生效
- 或使用设置面板点击「重新加载配置」

### 5.4 配置验证

如果配置文件格式错误：
- 应用会自动使用默认配置
- 在设置面板中会显示错误提示
- 建议备份后删除 `config.json` 重新生成

## 七、常见问题

### Q1: PrivacyGuard 是免费的吗？

**A**: 是的，PrivacyGuard 完全免费使用，没有功能限制。

### Q2: 脱敏后的文件会保存到云端吗？

**A**: 不会。所有处理都在本地完成，文件不会上传到任何服务器，确保您的数据安全。

### Q3: 支持批量处理多个文件吗？

**A**: 当前版本支持逐个文件处理。批量处理功能将在未来版本中添加。

### Q4: 可以自定义替换文本吗？

**A**: 可以。通过修改 `config.json` 中的 `redaction.replacement_text` 配置项。

### Q5: 如何处理识别不准确的 PDF？

**A**:
1. 在 OCR 设置中选择更高的精度级别（3.0x）
2. 手动添加或调整脱敏框位置
3. 使用偏移调整功能微调位置

### Q6: 支持哪些语言的 OCR 识别？

**A**: 主要支持中文和英文。其他语言的识别效果可能不理想。

### Q7: 可以在移动设备上使用吗？

**A**: 当前仅支持 macOS 和 Windows 桌面版。移动端版本正在规划中。

### Q8: 如何反馈问题或建议？

**A**: 点击主界面的「反馈」按钮，填写问题描述后提交。

## 八、故障排除

### 8.1 应用无法启动

**现象**：双击图标后没有反应

**解决方案**：
1. 检查系统版本是否符合要求
2. macOS：检查是否已授权打开（系统设置 → 隐私与安全性）
3. Windows：尝试以管理员身份运行
4. 检查日志文件 `logs/privacyguard.log`

### 8.2 PDF 打开失败

**现象**：提示「无法打开 PDF」或空白页面

**解决方案**：
1. 确认文件不是损坏的
2. 尝试用其他 PDF 阅读器打开验证
3. 检查文件是否被其他程序占用
4. 尝试重启应用

### 8.3 OCR 识别结果不准确

**现象**：识别出的文字有误或位置偏移

**解决方案**：
1. 提高 OCR 精度设置（设置 → 扫描精度）
2. 手动添加遗漏的脱敏框
3. 使用偏移调整功能修正位置
4. 对于手写体或艺术字体，识别效果可能不佳

### 8.4 Word 格式错乱

**现象**：Word 文档打开后格式与原文件不同

**解决方案**：
1. 这是正常现象，部分复杂格式可能无法完全保留
2. 建议在脱敏后检查文档格式
3. 如有严重问题，可尝试精确模式单独处理

### 8.5 脱敏后文件保存失败

**现象**：点击保存后没有文件生成

**解决方案**：
1. 检查保存路径是否有写入权限
2. 尝试保存到其他位置（如桌面）
3. 检查磁盘空间是否充足
4. 确保文件名不含特殊字符

### 8.6 配置文件错误

**现象**：提示「配置文件格式错误」

**解决方案**：
1. 备份并删除 `config.json`
2. 重启应用，系统会生成默认配置
3. 参考 `config.json.template` 重新配置
