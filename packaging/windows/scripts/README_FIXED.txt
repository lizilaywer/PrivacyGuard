================================================================================
                    PrivacyGuard Windows 打包指南 (已修复)
================================================================================

本目录包含的脚本已修复，可在 Windows 上正常运行。

修复的问题：
1. UTF-8 编码问题 - 改为使用系统默认代码页
2. 路径处理 - 添加引号处理包含空格的路径
3. 目录切换检查 - 添加错误处理
4. version.txt 空行问题 - 添加跳过空行逻辑
5. Inno Setup 路径查找 - 支持多个常见安装位置
6. 文件存在检查 - 复制文件前检查是否存在

================================================================================
                            使用步骤
================================================================================

第一次打包（按顺序执行）：

    1. 双击运行 "1_初始化环境.bat"
       等待几分钟，安装依赖

    2. 双击运行 "3_完整打包带安装程序.bat"
       等待 5-10 分钟

    3. 在 releases\windows 文件夹中找到安装程序
       PrivacyGuard-36.4-Setup.exe

================================================================================
                            问题排查
================================================================================

如果脚本仍然无法运行：

1. 检查文件编码：
   - 右键点击 bat 文件 -> 编辑
   - 确保文件是 ANSI 编码（不是 UTF-8）

2. 检查权限：
   - 右键点击 bat 文件 -> 以管理员身份运行

3. 手动检查路径：
   - 确保 packaging/windows/scripts/ 在项目根目录的正确位置
