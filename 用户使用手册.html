快速入门

提示

首次使用建议先用测试文件熟悉操作流程，然后再处理正式文档。

PDF 脱敏流程

1
打开 PDF 文件

拖拽 PDF 到窗口，或点击「选择文件」。系统自动检测 PDF 类型（文字版/图片版）

2
自动识别

系统自动识别敏感信息，用红色方框标记检测区域

3
调整脱敏区域

添加、删除或调整脱敏框位置和大小，确保覆盖所有敏感信息

4
执行脱敏

点击「开始脱敏」，选择保存位置，等待处理完成

Word 脱敏流程

精确模式

精确控制

选中需要脱敏的文字 → 右键「脱敏选中内容」→ 文字替换为 [已脱敏]

全局模式

自动扫描

点击「扫描敏感信息」→ 系统自动高亮所有匹配项 → 逐条确认或一键全部脱敏

PDF 处理功能

OCR 文字识别

对于图片版 PDF（扫描件），系统使用 OCR 技术识别文字：

高速度

适合清晰文档，处理速度最快

平衡模式

推荐设置，速度与精度兼顾

高精度

适合模糊文档，识别准确率最高

配置说明

配置文件位置

PrivacyGuard/
├── PrivacyGuard.exe (Windows) 或 PrivacyGuard.app (macOS)
├── config.json              ← 用户配置文件（自动生成）
└── config.json.template     ← 配置模板参考

常用配置项

脱敏规则

{
  "redaction": {
    "default_rules": {
      "身份证号": {
        "pattern": "(?<!\\d)([1-9]\\d{5}(19|20)\\d{2}...",
        "enabled": true
      },
      "手机号": {
        "pattern": "(?<![\\d])(1[3-9]\\d{9})(?![\\d])",
        "enabled": true
      }
    },
    "replacement_text": "[已脱敏]"
  }
}

OCR 设置

{
  "ocr": {
    "accuracy": {
      "low": 1.5,
      "medium": 2.0,
      "high": 3.0,
      "default": 2.0
    }
  }
}
配置热重载

修改 config.json 后，重启应用生效，或在设置面板点击「重新加载配置」。

常见问题

PrivacyGuard 是免费的吗？

是的，PrivacyGuard 完全免费使用，没有功能限制。

脱敏后的文件会保存到云端吗？

不会。所有处理都在本地完成，文件不会上传到任何服务器，确保您的数据安全。

可以自定义替换文本吗？

可以。通过修改 config.json 中的 redaction.replacement_text 配置项。

如何处理识别不准确的 PDF？

1. 在 OCR 设置中选择更高的精度级别（3.0x）
2. 手动添加或调整脱敏框位置
3. 使用偏移调整功能微调位置

支持哪些语言的 OCR 识别？

主要支持中文和英文。其他语言的识别效果可能不理想。

如何反馈问题或建议？

点击主界面的「反馈」按钮，填写问题描述后提交，或发送邮件至 491445490@qq.com

故障排除

应用无法启动

解决方案

1. 检查系统版本是否符合要求
2. macOS：检查是否已授权打开
3. Windows：尝试以管理员身份运行
4. 检查日志文件 logs/privacyguard.log

OCR 识别结果不准确

1
提高 OCR 精度

设置 → 扫描精度 → 选择「高精度」

2
手动添加

手动添加遗漏的脱敏框

3
偏移调整

使用偏移调整功能修正位置

配置文件错误

修复方法

1. 备份并删除 config.json
2. 重启应用，系统会生成默认配置
3. 参考 config.json.template 重新配置
