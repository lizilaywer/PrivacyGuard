# PrivacyGuard 下一步计划

> 待办事项和开发路线图

## 当前优先级

### 高优先级
- [ ] Windows 平台测试和优化
- [ ] 大文件性能优化（50+ 页文档）

### 中优先级
- [ ] 精确模式稳定性改进（偶尔失败问题）
- [ ] .doc 格式转换优化

### 低优先级
- [ ] 代码重构（模块化拆分）
- [ ] 单元测试覆盖

## 功能规划

### 短期（v37.x）
1. Windows 深色模式完全支持
2. 批量文件处理
3. 自定义规则增强

### 中期（v38.x）
1. 性能优化：大文件加载
2. 导出格式扩展（图片、纯文本）
3. 撤销/重做历史记录

### 长期（v40+）
1. 多语言支持
2. 插件系统
3. 云端规则同步

## 已知问题

| 问题 | 影响 | 状态 |
|------|------|------|
| 精确模式偶发失败 | <5% 选择失败 | 需改进 |
| 大文档延迟 | 50+ 页 <15s | 可接受 |
| .doc 格式转换 | 依赖外部工具 | 低优先级 |

## 贡献指南

1. 创建备份：`cp main.py backups/v36/main.py.backup_$(date +%Y%m%d_%H%M%S)`
2. 修改代码后运行语法检查
3. 更新 `docs/current/DEV_LOG.md`
4. 测试主要功能
