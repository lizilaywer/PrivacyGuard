# PrivacyGuard v22.1 测试计划

**测试日期:** 2026-02-09
**项目版本:** v22.1 - Viewport Filter
**修复内容:** 将事件过滤器从 `canvas_container` 移动到 `scroll.viewport()`

---

## 🔧 修复内容

### 问题诊断
- **原问题**: 右键点击完全无效，没有 [DEBUG] 日志输出
- **根本原因**: QScrollArea 的 `viewport()` 控件拦截了鼠标事件，导致事件过滤器无法捕获
- **解决方案**: 将事件过滤器安装到 `scroll.viewport()` 上

### 代码变更

#### 变更 1: 事件过滤器安装位置 (line 652)
```python
# 修改前 (v22.0)
self.canvas_container.installEventFilter(self)

# 修改后 (v22.1)
self.scroll.viewport().installEventFilter(self)
```

#### 变更 2: 事件过滤器对象检查 (line 520)
```python
# 修改前 (v22.0)
if obj == self.canvas_container and event.type() == event.Type.MouseButtonPress:

# 修改后 (v22.1)
if obj == self.scroll.viewport() and event.type() == event.Type.MouseButtonPress:
```

#### 变更 3: 版本号更新 (line 24)
```python
# 修改前
VERSION = "22.0 - Container Monitor"

# 修改后
VERSION = "22.1 - Viewport Filter"
```
