## ⚠️ 故障排除

### 问题1：右键点击仍然无响应
**可能原因：**
- 事件过滤器未正确安装
- 防抖时间设置过长

**解决方法：**
```bash
# 查看日志确认事件过滤器是否工作
grep "eventFilter" app_debug.log

# 如果没有输出，检查代码
grep "installEventFilter" main.py
```

### 问题2：样式警告仍然存在
**可能原因：**
- 缓存问题
- 其他按钮仍在使用旧样式

**解决方法：**
```bash
# 搜索 filter 属性
grep -n "filter:" main.py

# 应该没有输出（或者只在注释中）
```

### 问题3：左键点击仍然重复
**可能原因：**
- 防抖时间过短
- 事件仍然被父控件处理

**解决方法：**
- 调整 `_event_debounce_ms` 的值
- 检查父控件的事件处理
